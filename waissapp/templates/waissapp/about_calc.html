Net Opportunity Time (Tn)

Tn=((Fn-c)/a))^(1/b)
Where:
Tn = net opportunity time (min)
Fn = net depth of application (mm)
a, b, c = infiltration constants (see Table 1)

Net Application Depth (Fn) 〖 F〗_n=(FC-PWP)*MAD*DRZ Where: FC = field capacity, by volume PWP= permanent wilting point, by volume DRZ= depth of rootzone (m) MAD= management allowable deficit Advanced Time (Tt) - time required for water to cover the field 〖 T〗_t=T_n*R Where: Tt = advance time (min) Tn = net opportunity time (min) R = efficiency-advance ratio (see Table 2) Basin Length (L) L=(6x10^4 Q_u T_t)/((〖aT_t〗^b/(a+b)+7)+(1798n^(3/8) 〖Q_u〗^(9/16) 〖T_t〗^(3/16) ) Where: L = basin length (m) Qu= unit inflow rate (m3/s/m) n = Manning’s roughness coefficient Ea= Application Efficiency Inflow time (Ta) T_a=FnL/(600Q_u E_a ) Where: Ta = inflow time (min) Where: Ea=F_n/F_g Design Rule: T can be either Ta or Tt, whichever is greater.
